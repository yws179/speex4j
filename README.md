# Speex4J (SPEEX FOR JAVA)

![](https://img.shields.io/github/license/yws179/speex4j)

### 使用步骤

1. 安装 `Speex` 库

[前往Speex 官网下载](https://www.speex.org/)


2. 拉取本仓库代码
```shell
git clone git@github.com:yws179/speex4j.git    
```

3. Linux 下编译so库

```shell
# 进入项目目录
cd speex4j/native
# 生成so库
./script/build_share_lib.sh
```
